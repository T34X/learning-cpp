# Making a simple web server using Crow(C++).
Installation:
https://crowcpp.org/master/
</br>
### Usage
At first compile the program:
```
g++ -o server main.cpp
```
Then run as root:
```
sudo ./server
```

More examples on <a href="https://github.com/CrowCpp/Crow/tree/master/examples">official page.</a>
